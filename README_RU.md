# Образовательная платформа

Современная платформа для онлайн-обучения с поддержкой курсов, тестирования и сертификации.

## Возможности

### Курсы
- Полные CRUD операции для курсов
- Связь с преподавателями
- Категории и уровни сложности
- Рейтинги и отзывы
- Прогресс студентов

### Тестирование
- Различные типы вопросов (одиночный выбор, множественный выбор, истина/ложь, эссе)
- Автоматическая проверка ответов
- Результаты с подробной обратной связью
- Несколько попыток с отслеживанием лучших результатов

### Сертификация
- Генерация PDF сертификатов
- Уникальные ID и коды проверки
- Связь с пользователями и курсами
- Проверка валидности через API
- Автоматическое создание при завершении курса

## Технологии

- **Node.js** - серверная платформа
- **Express** - веб-фреймворк
- **MongoDB** - база данных
- **Mongoose** - ODM
- **PDF-lib** - генерация PDF
- **JWT** - аутентификация

## Установка

1. Клонируйте репозиторий:
```bash
git clone <url>
cd plathorm_lessons
```

2. Установите зависимости:
```bash
npm install
```

3. Создайте файл `.env` и настройте переменные:
```
NODE_ENV=development
PORT=5000
MONGODB_URI=mongodb://localhost:27017/learning-platform
JWT_SECRET=your_jwt_secret_key_here
CLIENT_URL=http://localhost:3000
```

4. Запустите сервер:
```bash
npm run dev
```

## Структура API

### Курсы
- `GET /api/courses` - получить все курсы
- `GET /api/courses/:id` - получить курс
- `POST /api/courses` - создать курс
- `PUT /api/courses/:id` - обновить курс
- `DELETE /api/courses/:id` - удалить курс
- `POST /api/courses/:id/enroll` - записаться на курс
- `DELETE /api/courses/:id/unenroll` - отменить запись
- `PUT /api/courses/:id/progress` - обновить прогресс

### Тестирование
- `POST /api/tests/start/:quizId` - начать тест
- `GET /api/tests/:id` - получить детали теста
- `POST /api/tests/:id/answer` - отправить ответ
- `POST /api/tests/:id/complete` - завершить тест
- `GET /api/tests/:id/results` - получить результаты
- `GET /api/tests/history/:quizId` - история тестов

### Сертификация
- `POST /api/certificates/generate` - создать сертификат
- `GET /api/certificates/my-certificates` - получить мои сертификаты
- `GET /api/certificates/:id` - получить сертификат
- `GET /api/certificates/download/:id` - скачать PDF
- `GET /api/certificates/verify/:certificateId` - проверить сертификат
- `GET /api/certificates/course/:courseId` - сертификаты курса

## Безопасность

- JWT аутентификация
- Контроль доступа по ролям (студент, преподаватель, администратор)
- Санитизация ввода
- Ограничение частоты запросов
- Защита от XSS и других атак
